# 🚀 FlowPilot

FlowPilot is a modern workflow and productivity management platform built using Python, Streamlit, and MySQL.

It helps users organize tasks, manage workflow, track pending/completed work, and improve productivity using a clean dashboard interface.

---

# ✨ Features

- Add Tasks
- Complete Tasks
- Delete Tasks
- Task Filtering
- Dashboard Metrics
- Modern Dark UI
- MySQL Database Integration
- CRUD Operations

---

# 🛠 Tech Stack

- Python
- Streamlit
- MySQL
- SQL
- CSS
